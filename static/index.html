Welcome to CNC.js!

If you see this page, the mount point for serving static files is working. Further configuration is required.

  1. Run cnc with -h for detailed usage:
    $ cnc -h

  Usage: cnc [options]
  
  Options:
    -m, --mount [<url>:]<path>  set the mount point for serving static files (default: /static:static)
  2. Specify a mount path for the static directory, as shown below:
    $ cnc -m /static:/path/to/your/static/files

For online documentation please refer to https://github.com/cheton/cnc.js/wiki.
